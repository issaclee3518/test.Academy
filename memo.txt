git 버전 확인
git --version

프로젝트 디렉토리에 로컬 저장소 생성
git init

git 버전 관리할 파일을 선택(선택만): stage에 올림
git add test.py

버전 만들기 : 커밋
git commit -m "test.py커밋"(메시지)

로그 확인
git log

로컬 저장소에서 github 원격 저장소 주소를 등록
git remote add origin https://github.com/issaclee3518/test.Academy.git

생성한 커밋들을 github에 전송(push)
git push -u origin main

Fork
하나의 저장소를 다른 계정으로 복사
git 명령어가 아님(github 기능)
다른 사람의 github 계정 저장소에서 fork를 눌러 내 계정 저장소로 복사
